			Tema 2 SO - Memory Allocation Library
	
-------------------------------------------------------------------------------	

	Minimalistic memory allocator that can be used to manually manage 
	virtual memory, that accounts for explicit allocation, 
	reallocation, and initialization of memory.

	It uses basics of memory management by implementing minimal 
	versions of `malloc()`, `calloc()`, `realloc()`, and `free()` 
	and emory management syscalls in Linux: `brk()`, `mmap()`
	and `munmap()`.
